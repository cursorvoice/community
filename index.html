How it works

1

Describe a tool

A plugin is one JSON file — a name, a description, and an action: open a URL, run a shell command, or AppleScript.

2

Drop it in

Save it to ~/Library/Application Support/CursorVoice/plugins/. Cursor Voice picks it up automatically.

3

Just say it

The assistant can now call your tool by voice. Use Submit to share it here for everyone.
